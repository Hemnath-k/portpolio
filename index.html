STUDENT GRADE CALCULATOR IN JAVA

This Java program calculates and displays a student's average grade and corresponding letter grade based on scores entered for multiple subjects.
Users input the number of subjects and their grades, and the program computes the average.
It then assigns a letter grade based on predefined thresholds (A, B, C, D, F) and outputs the results.
